Практическая часть

Разработайте три страницы по заданию: главную index.html, страницу page1.html и страницу page2.html. Страницы делайте по образцам из папки templates.

Не требуется фотографического воспроизведения страниц, используйте только те возможности, которые были использованы в примерах из материалов занятия.

Страницы page1.html и page2.htmlразместите в папке htdocs. На главной странице разместитетекст из практической части этого задания.

В верхней части каждой страницы разместите навигационные ссылки для перехода на все страницы.

В нижней части каждой страницы выводите данные о разработчике, название группы, города, год разработки.

Для страницы page2.html реализуйте якоря внутри страницы, навигацию по этим якорям.
